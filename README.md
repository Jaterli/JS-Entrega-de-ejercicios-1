## 📋 Descripción

Este repositorio contiene 9 ejercicios prácticos de JavaScript desarrollados como parte del Máster en Desarrollo Web Full Stack. Cada ejercicio demuestra diferentes conceptos fundamentales de JavaScript, desde manipulación básica del DOM hasta persistencia de datos con localStorage.

## 🚀 Ejercicios

### 1. Cambiador de Color de Fondo
**Objetivo:** Practicar eventos en JavaScript y manipulación de estilos del DOM.

- Botón que cambia el color de fondo a un color aleatorio
- Demuestra el uso de `addEventListener` y manipulación de estilos CSS

### 2. Contador de Clics
**Objetivo:** Practicar el manejo de eventos y actualización del contenido del DOM.

- Botón que incrementa un contador
- Actualización en tiempo real del número de clics y de un mensaje de animación

### 3. Lista Dinámica
**Objetivo:** Trabajar con creación, eliminación y manipulación de elementos del DOM.

- Campo de texto para agregar elementos
- Cada elemento tiene su propio botón de eliminación
- Demuestra creación dinámica de elementos HTML

### 4. Filtro de Búsqueda en Tiempo Real
**Objetivo:** Practicar eventos del DOM y lógica de filtrado.

- Lista predefinida de elementos
- Campo de búsqueda que filtra en tiempo real
- Demuestra el uso del evento `input` y filtrado de arrays

### 5. Calculadora Sencilla
**Objetivo:** Practicar manipulación de formularios, eventos y lógica básica.

- Dos campos de entrada numéricos
- Cuatro operaciones básicas (sumar, restar, multiplicar, dividir)
- Validaciones para evitar división por cero y entradas vacías

### 6. Temporizador
**Objetivo:** Practicar funciones de temporización y manipulación del DOM.

- Formato HH:MM:SS
- Botones para iniciar, pausar y reiniciar
- Demuestra el uso de `setInterval` y `clearInterval`

### 7. Generador de Contraseñas Aleatorias
**Objetivo:** Practicar generación de cadenas aleatorias y validación de formularios.

- Campo para especificar longitud
- Generación con letras, números y caracteres especiales
- Validación de longitud mínima

### 8. Contador de Palabras y Caracteres
**Objetivo:** Practicar eventos en tiempo real y manipulación avanzada del DOM.

- Área de texto para escribir párrafos
- Conteo en tiempo real de palabras y caracteres (sin espacios)
- Demuestra el uso de expresiones regulares

### 9. Lista de Tareas con LocalStorage
**Objetivo:** Practicar persistencia de datos con localStorage.

- Crear y eliminar tareas
- Marcar tareas como completadas
- Limpiar todas las tareas completadas
- Persistencia de datos al recargar la página

## ✨ Características Destacadas

- ✅ **Código limpio y comentado** - Fácil de entender y mantener
- ✅ **Validaciones completas** - Manejo de errores en todos los ejercicios
- ✅ **Persistencia de datos** - Las tareas se guardan en localStorage
- ✅ **Diseño consistente** - Estilos unificados con CSS
- ✅ **Navegación intuitiva** - Enlaces de regreso en todas las páginas
- ✅ **Sin dependencias externas** - JavaScript puro, sin frameworks
- ✅ **HTML semántico** - Uso de etiquetas como `<header>`, `<main>`, `<section>`, `<article>`, `<nav>`
- ✅ **Accesibilidad mejorada** - Implementación de atributos ARIA (aria-label, aria-labelledby, aria-hidden) para lectores de pantalla
